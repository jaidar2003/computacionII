
MacOS
=====

# Iniciar el servidor de archivos
python /Users/juanmaaidar/PycharmProjects/computacionII/final/servidorArchivos/main.py -m server

# O iniciar solo la API Flask (requiere que el servidor de archivos esté en ejecución)
python /Users/juanmaaidar/PycharmProjects/computacionII/final/servidorArchivos/main.py -m api -H 192.168.100.146 -p 5005

# Iniciar el frontend
cd final/servidorArchivos/front
# Editar el archivo .env para configurar VITE_API_URL si la API está en otra máquina
npm run dev
